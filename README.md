QR Code Generator
This repository provides a simple Python tool to generate QR code images from URLs using the qrcode library.

# Features
Automated Generation: Hardcoded to generate a specific LinkedIn profile QR code instantly.

Interactive Function: Includes qr_code_gen() for custom user input.

PNG Export: Automatically saves generated codes in high-quality image format.

# Installation
To get started, install the required library:

Bash
pip install qrcode[pil]
# Usage Instructions
Run the script: Execute the Python file in your terminal.

Provide Link: Enter the full URL when prompted.

Name the File: Type your preferred filename (the script adds .png automatically).
